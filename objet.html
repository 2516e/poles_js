Une variable de type object va être constitué de :

  • propriétés : ce sont des valeurs qui sont liées à l'objet.
  • méthodes : ce sont des fonctions qui sont liées à l'objet.
Pour accéder aux propriétés ou aux méthodes d'un objet, on utilise le point .

Par exemple, push est une méthode d'un objet de type Array.

En JS, les strings sont des objets.
