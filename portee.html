Portée des variables (=scope)

  • Une variable déclarée dans une fonction n'est accessible que dans cette fonction. On dit que c'est une variable locale à la fonction.
  • Une variable déclarée avec le mot clé var est une variable globale. Elle est accessible partout.
  • Une variable déclarée avec le mot-clé let n'est accessible que dans le block où elle a été déclarée. Un bloc est délimité par des {}.
    On ne peut pas redéclarer une variable déclarée avec let dans le même bloc.
  • Le mot-clé const permet de déclarer des constantes. Une constante est une variable dont la valeur ne peut pas être modifiée.
    Comme pour let, const a une portée de bloc.
